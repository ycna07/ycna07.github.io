安装ArchWSL#

直接从微软商店安装即可，没有使用ArchWSL项目

或者

wsl -d Arch

开启systemd服务#

vim /etc/wsl.conf

写入

[boot]
systemd=true
  • 全局开启镜像网络模式

目前微软官方更加推荐使用 镜像网络模式

打开pwsh

cd ~
code .wslconfig

写入

[wsl2]
networkingMode = mirrored
