reference 菜鸟上的Git教程

查看暂存区

git add filename       # 将单个文件添加到暂存区
git add .              # 将工作区中的所有修改添加到暂存区
git status             # 查看哪些文件在暂存区中

git rm 删除git跟踪的文件

git删除文件并推送到repo流程

git rm <filename>
git commit -m 'delete filename'
git push

git diff
